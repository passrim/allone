적립방법

  • 올원뱅크 최초 로그인
    올원뱅크 가입 후 최초 로그인 하시면 1회에 한하여 50캔디 적립
    (※ 기존 사용자가 올원뱅크 회원탈퇴 후 재가입의 경우 해당되지 않음)
  • 최초 계좌 등록
    올원뱅크 계좌 등록 시 1회에 한하여 30 캔디 적립
    (※ 기존 사용자가 올원뱅크 회원탈퇴 후 재가입-재등록의 경우 해당되지 않음)
  • 입출금계좌개설
    입출금계좌개설 시 1회에 한하여 30 캔디 적립
  • 올원송금
    • 계좌송금 시 일 4회(거래완료 건)에 한하여 건 별 5 캔디 적립
    • 휴대폰송금 시 일2회(거래완료 건)에 한하여 건 5 캔디 적립
  • 환전 및 환전 선물 시
    환전 및 환전 선물 시 월 2회(거래완료 건)에 한해서 건 별 5 캔디 적립
  • 상품가입 시
    신규로 상품가입 시 월(3회)에 한하여 20 캔디 적립
    (※ 대출관련 상품 제외)
  • 로그인 시
    로그인 시 일(1회)에 한하여 2 캔디 적립
  • 모임 참여
    • 모임개설 시 일(3회)에 한하여 20 캔디 적립
    • 모임참여 시 일(6회)에 한하여 5 캔디 적립
