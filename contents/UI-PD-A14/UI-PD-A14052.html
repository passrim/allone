  • 꼭 알아두세요!
    • 해외송금 미리메모 등록서비스(Remittance information pre-registration service)란?
      • 해외송금정보(수취인정보)를 송금신청 전에 미리 등록하실 수 있는 서비스로, 해외송금 신청 시 간편하게 메모내역 불러오기를 통하여 송금처리를 할 수 있습니다.
      • 무계좌해외송금 미리메모 서비스는 수취인의 해외은행 계좌번호를 아실 때 사용 가능합니다.
      • 서비스 이용 가능 시간 : 24시간 365일
      • 서비스 이용 채널 : 인터넷뱅킹,올원뱅크, 영업점(등록, 변경, 삭제, 조회 가능, 단, 스마트뱅킹은 메모내역 확인 및 불러오기만 가능)
      • 미리메모 서비스로 등록하신 정보의 유효기간은 등록일로부터 1년이며, 유효기간 중에만 연장등록이 가능합니다.
      • 유효기간 경과 후 해당 정보는 고객님의 정보보호를 위해 자동 삭제됩니다.
