위드(WITH) 번호

위드(WITH)포인트 란?

  • 고객님이 가입한 대출상품을 다른 고객에게 소개하여 가입하게 한 경우 농협 채움포인트를 적립해 드리는 서비스입니다.
    • 적립기준 : 대출금액 500만원 이상으로 대출신청 시 위드(With)번호를 입력한 경우
    • 적립한도 : 최고 20,000p(소개한 포인트, 소개받은 포인트 각 10,000p)
    • 적립시기 : 전월말 기준 익월 10일
  • NH채움카드를 소지한 경우에만 포인트가 적립됩니다.
  • 위드(With)번호는 대출 계좌번호(13자리 숫자)입니다.
  • 적립시기에 적립기준을 미충족하는 경우 소급하여 처리되지 않습니다.
