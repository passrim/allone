  • 도로명(신주소) 검색방법 : 도로명(~로, ~길) + 건물번호
    예) 서울시 강동구 양제대로 112길 57 > 양재(도로명)57(건물번호)
  • 동명(구주소) 검색방법 : 동명(~동)+번지
    예) 서울시 강남구 대치동 928-10 > 대치동(동명) 928(번지)
  • 건물명(신주소) 검색방법 : 건물명
