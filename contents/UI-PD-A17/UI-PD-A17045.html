앱 접근 권한 안내

올원뱅크 이용을 위해 다음 접근권한 허용이 필요합니다.

선택적 접근 권한 항목은 접근권한을 허용하지 않아도 서비스 이용은 가능하나, 일부 기능 사용에 제한이 있을 수 있습니다.

선택적 접근 권한

  • [선택] 주소록 올원송금, 환전송금 등에서 상대방의 연락처 정보를 사용하기 위해 접근합니다.
  • [선택] 카메라 입출금계좌개설 시 신분증촬영, 영상통화 인증 및 QR결제 및 Face ID 이용 시 사용됩니다.
  • [선택] 사진 마이올원 이미지 등록, 초대장 이미지, 자주쓰는 송금 이미지 등록, 올원커뮤니티 글쓰기, 올원 프렌즈 다운로드, 영수증 저장 시 사용합니다.
  • [선택] 위치 위치기반으로 영업점 찾기 등을 위해 사용합니다.
  • [선택] 마이크 음성송금서비스, 음성메뉴이동서비스, 입출금계좌개설 영상통화 인증 이용 시 사용합니다.
  • [선택] 음성인식 음성송금서비스, 음성메뉴이동 서비스 이용 시 사용합니다.
  • [선택] 블루투스 입출금계좌개설 시 영상통화 인증이용 시 사용됩니다.

접근권한 설정안내

디바이스별 ‘설정>개인정보보호’ 메뉴에서 설치된 앱의 접근권한을 동의하거나 철회할 수 있습니다.
