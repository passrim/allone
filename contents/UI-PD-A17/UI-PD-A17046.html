앱 접근 권한 안내

올원뱅크 이용을 위해 다음 접근권한 허용이 필요합니다.

필수적 접근 권한 항목은 올원뱅크 앱의 실행을 위해 반드시 필요하며 허용하지 않을 경우 서비스 이용이 제한됩니다.

선택적 접근 권한 항목은 접근권한을 허용하지 않아도 서비스 이용은 가능하나, 일부 기능 사용에 제한이 있을 수 있습니다.

필수적 접근 권한

  • [필수] 저장공간 앱 설치 및 공인인증서, 앱의 콘텐츠와 이미지 저장을 위해 사용합니다.
  • [필수] 전화 전자금융 사기예방 등을 위해 기기 정보를 확인합니다.
  • [필수] SMS SMS 인증 시 인증번호 자동 확인을 위해 사용됩니다.

선택적 접근 권한

  • [선택] 주소록 올원송금, 환전송금 등에서 상대방의 연락처 정보를 사용하기 위해 접근합니다.
  • [선택] 카메라 입출금계좌개설 시 신분증촬영, 영상통화 인증 및 QR결제 이용 시 사용됩니다.
  • [선택] 위치 위치기반으로 영업점 찾기 등을 위해 사용합니다.
  • [선택] 마이크 음성송금서비스, 음성메뉴이동서비스, 입출금계좌개설 영상통화 인증 이용 시 사용합니다.
  • 안드로이드OS 6.0미만 버전 스마트폰을 이용하시는 경우 선택적 접근권한 없이 모두 필수적 접근권한으로 적용될 수 있습니다.

접근권한 설정안내

설정> 애플리케이션 관리자> 올원뱅크> 권한
