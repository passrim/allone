스마트기기에서 스마트기기로 공인인증서 내보내기

다른 스마트 기기에 저장되어 있는 인증서를 스마트폰으로 복사하여 다른 스마트기기와 동일한 인증서를 사용합니다.

  • 스마트 기기
  • 스마트 기기
  • 인증서 복사 안내
    1
    현재 스마트기기에서 다른 스마트기기로 내보낼 공인인증서를 선택
    2
    공인인증서 암호를 입력
    3
    다른 스마트기기에서 농협 스마트뱅킹 혹은 올원뱅크 앱을 실행
    4
    해당 기기에서 인증서 가져오기 메뉴 선택
    • - 스마트뱅킹 : 인증센터 > 인증서 내보내기 > 스마트폰→스마트폰
    • - 올원뱅크 : 고객센터 > 인증서 복사 > 스마트폰으로 인증서 가져오기
    5
    이 스마트기기에 나타나는 인증번호를 입력하고 다음으로 진행
    6
    공인인증서 암호를 입력
    7
    인증서 복사 완료
