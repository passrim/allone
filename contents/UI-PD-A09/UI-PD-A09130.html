더치페이 안내

  • 총 금액/명수만 입력하면 자동으로 1/n 계산은 물론, 계좌번호 입력 없이 간편하게 송금까지 가능한 서비스입니다.

올원뱅크 로그인 후, 더치페이 메뉴 클릭!
더치페이를 할 그룹명을 지정한 후 총 금액 및 참석자 수를 입력하세요. 그러면 계산기 사용 필요 없이 자동으로 1/n금액이 나온답니다.

차수추가를 선택하면 2차, 3차 각 차수 별 금액계산도 문제없어요. (10차까지 추가 가능합니다.)

입금 받을 계좌번호를 선택하고 메시지를 보낼 SMS 혹은 카카오톡을 선택해 주세요. 그리고 문자 발송!

카카오톡 또는 SMS로 받는 분에게 더치페이를 바로 요청 할 수 있는 매우 편리한 서비스입니다.
