올원송금 안내

  • 휴대폰번호 또는 계좌번호로 일 300만원까지 송금가능하며 수수료가 면제되는 편리한 서비스입니다.
    (※ 50만원 이상 타행송금 시 수수료가 발생 할 수 있습니다.)

출금계좌를 선택한 후,
송금할 계좌 또는 내 연락처에서 송금받을 분의 실명과 휴대폰 번호를 선택하세요.

송금하실 금액을 입력 하신 후,
송금하기 버튼을 선택하면 바로 송금이 됩니다.

카카오톡 또는 SMS로 받는 분에게 송금내역을 바로 전달 할 수 있으며, 한번 송금한 내역은 자주찾는 송금으로 등록하여 송금 정보를 입력해야하는 번거로움 없이 바로송금이 가능합니다.
