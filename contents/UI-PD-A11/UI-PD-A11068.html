  • 농협계좌 등록 계좌정보 한번만 등록하고 안전하게 이용하세요!
  • 꼭 알아두세요!
    • 청구서
      카톡에서 확인하고, 사용내역은 알기 쉽게, 연체료 걱정 없이 미리 알려드립니다.
    • 인증
      별도 앱 설치없이, 비밀번호만으로 인증, 인증 홈에서 인증 내역 확인까지 할 수 있습니다.
    • 송금
      카톡친구라면 누구나, 채팅방에서 바로 송금, 지문 혹은 비밀번호 하나만 있으면 가능합니다.
    • 결제
      카톡 이용자라면 비밀번호로 바로 결제! 보안 걱정 없이 이용하세요.
    • 멤버십
      나만의 멤버십 바코드에 다양한 멤버십을 담고 바코드 하나로 포인트 적립/사용이 가능합니다.
